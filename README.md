# CarbonCast

CarbonCast is a modern, enterprise-grade climate-tech web application that helps individuals understand, monitor, and reduce their environmental impact through intelligent carbon footprint estimation and actionable sustainability insights.

## Project Structure

This project follows a monorepo structure:
- `frontend/`: React (Vite) application using Tailwind CSS.
- `backend/`: Node.js/Express RESTful API with MongoDB.

## Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB

### Backend Setup
1. Navigate to the `backend` directory: `cd backend`
2. Install dependencies: `npm install`
3. Create a `.env` file based on `.env.example`.
4. Start the server: `npm run dev`

### Frontend Setup
1. Navigate to the `frontend` directory: `cd frontend`
2. Install dependencies: `npm install`
3. Create a `.env` file if needed based on `.env.example`.
4. Start the development server: `npm run dev`

## Features
- Multi-step Carbon Footprint Calculator
- Personalized Dashboard & Recommendations
- Monthly Analytics & Progress Tracking
- Sustainability Tips & Actionable Insights
- Responsive and Accessible UI
